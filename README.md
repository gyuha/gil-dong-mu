# 무녀: 밤을 부르는 자

조선 시대, 천민으로 내몰린 무녀가 요괴와 원혼이 들끓는 밤을 동료들과 함께 살아남는 **동료 지휘형 호드 서바이벌 액션 로그라이트**입니다.

이 게임의 핵심 차별점은 주인공이 싸우지 않는다는 것입니다. 무녀는 직접 공격하지 않고 — 오라(감속·회복), 밀쳐내기, 혼불 전달, 명령만으로 자율 전투하는 동료들을 지휘해 밤을 넘깁니다. 모든 살상 화력은 동료가 냅니다.

> 세계관·시나리오 전문은 [`무녀 - 게임 컨셉 & 시나리오 초안.md`](<무녀 - 게임 컨셉 & 시나리오 초안.md>) 참고.

## 현재 상태

**그레이박스 프로토타입** (아트 없음, 도형만) — 코어 루프 검증 단계입니다.

- 한 밤(3분) 생존 루프: 잡귀·창귀 호드, 시간에 따른 스폰 강도 상승, 승패 판정·재시작
- 동료 2종(화랑 호위병 — 근접 탱커, 탈 쓴 퇴마사 — 근접 광역 베기)의 자율 전투와 명령 4종
- 혼불 전달형 성장: 무녀가 수집해 동료에게 전달하거나(근접 시), 곁에 동료가 없으면 3초 후 스스로 흡수 — "내 성장 vs 동료 성장"을 발로 결정
- 무녀·동료 각자의 레벨업 드래프트(일시정지 + 큐 일괄 선택)
- 동료 케어: 쓰러짐 → 근접 1.5초 구출, 실패 시 그 밤에서 이탈

## 실행

[Godot 4.6+](https://godotengine.org/) 필요.

```sh
godot          # 프로젝트 루트에서 에디터 실행 후 F5, 또는
godot scenes/main.tscn
```

### 조작

| 키 | 동작 |
|---|---|
| WASD / 방향키 | 무녀 이동 |
| Space | 밀쳐내기 (MP 30 소비, 주변 적을 밀어냄) |
| 1 | 명령: 주변에서 싸워 (기본 — 적극 교전) |
| 2 | 명령: 공격적으로 싸워 (무제한 추격) |
| 3 | 명령: 방어적으로 싸워 (소극 교전) |
| 4 | 명령: 모여라 (교전 중단, 무녀 곁 집결) |

## 테스트

GUT 비의존 자체 헤드리스 러너를 사용합니다.

```sh
# 단위 테스트 (순수 결정로직 — scripts/logic/)
godot --headless --script test/test_runner.gd

# 게임플레이 스모크 (키 입력 주입 자동 조종, s2~s7)
godot --headless --fixed-fps 60 --script test/smoke_s7.gd
```

순수 로직(타깃 선정, 혼불 분배, 드래프트 풀/큐, 명령 파라미터, 스폰 곡선, 승패 판정 등)은 테스트 우선(TDD)으로 작성합니다.

## 구조

```
scenes/main.tscn        메인 씬 (유일한 씬 — 엔티티는 스크립트로 생성)
scripts/                씬·노드 코드 (munyeo, companion, hwarang, exorcist, japgwi, changgwi, …)
scripts/logic/          노드 비의존 순수 결정로직 (전부 테스트됨)
test/                   test_*.gd 단위 테스트 + smoke_*.gd 게임플레이 스모크
.forge/                 개발 루프 문서 — CONTEXT.md(용어집), adr/(설계 결정), retro/(회고)
```

## 설계 문서

- [`.forge/CONTEXT.md`](.forge/CONTEXT.md) — 프로젝트 공용 언어(무녀, 동료, 혼불, 오라, 명령, 쓰러짐, 밤 …)
- [`.forge/adr/`](.forge/adr/) — 설계 결정 기록. 특히 [ADR-0003](.forge/adr/0003-support-only-munyeo.md): 무녀는 직접 공격하지 않는다 — 플레이 검증으로 내린, 장르 관례를 이탈하는 의도적 결정
