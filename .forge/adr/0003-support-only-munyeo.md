# 무녀는 직접 공격하지 않는다 — 서포트 전용 지휘자

코어 루프 그레이박스(task #1) UAT에서 사용자가 직접 플레이로 판정한 결정이다. 그릴링 단계에서는 VS류 관례와 컨셉 문서 16장("주술로 쓸어버리는 쾌감")을 근거로 무녀에게 자동 공격 주술을 부여했으나, 첫 플레이에서 번복되어 문서 8장 원안(순수 지휘자)으로 회귀했다. 무녀의 기여는 오라·밀쳐내기·혼불 전달·명령뿐이며, 모든 살상 화력은 동료가 낸다. 초반 쾌감은 동료 화력과 적 밀도 상향으로 이전한다.

플레이어 캐릭터가 공격하지 않는 호드 서바이벌은 장르 관례의 정면 이탈이므로, 장르 기본값으로 "고치려는" 시도를 막기 위해 기록한다. 무녀 드래프트 풀도 서포트 전용으로 재편된다(공격 강화 선택지 없음).
