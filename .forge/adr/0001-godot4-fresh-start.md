# 본 개발은 Godot 4로 백지에서 시작한다

게이트 프로토타입(커밋 1af0398, "자율 동료 지휘가 재밌는가" 검증용)은 **통과**로 결론났고, 이후 커밋 7e89d46에서 전체 삭제되었다. 본 게임 개발은 프로토타입 코드를 일절 재사용하지 않고 백지에서 시작하되, 엔진은 검증 경험이 축적된 **Godot 4**를 유지한다. Unity(DOTS의 대량 개체 처리 이점)를 고려했으나, 호드 서바이벌 2D는 단일 프로세스로 Godot 성능이 충분하고 라이선스·복잡도 비용이 없다는 이유로 기각했다.

프로토타입은 스스로 "일회용 검증 게이트"로 선언된 코드였으므로, 검증된 로직(스탠스·타깃팅·근접부활)이라도 가져오지 않는다 — 필요하면 git 히스토리(1af0398)에서 참조만 한다.
